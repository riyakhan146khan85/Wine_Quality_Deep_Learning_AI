# Wine_Quality_Deep_Learning_AI
An advanced multidimensional deeplearning pipeline constructed with google's **TensorFlow & Keras** frameworks to analyze complex non-linear chemical interactions and predict sensory output quality ratings.
## Core Engineering Achievements
* **Deep neural network architechture ** : Successfully engineered a three layer sequential artificial neural network ANN packed with **2,881 trainable parameter weights**
## Network layer architechture topology##
**Input Matrix**: 11 dimensions representing core chemical attributes
**Hidden Layer 1 (Dense)**: 64 neurons configured with non-linear ReLU activation functions.
**Hidden Layer 2 (Dense)**: 32 neurons capturing deep feature crosses and element interections.
**Output Node (Dense)**: 1 neuron outputting a continuous localised quality evalution score.
## Tracked chemical features
* Volatile & fixed acidity
* citric acid & residual sugar
* chlorides & density metrics
* free/total sulfur dioxide tracks
* pH and sulphate concentrations
